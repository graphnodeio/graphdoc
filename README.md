# graphdoc
Graphdoc helps you to deploy a fully featured graphql documentation portal with the following features
* Schema reference
* Guides
* Sample Collections
* Graphiql explorer

## Note: This project is still in development
The project is still in the early stages of development. Hoping to ship a decent usable version by end of Nov 2020.

Inspired by
* [Github docs](https://developer.github.com/v4/query/)
* [Gitlab docs](https://docs.gitlab.com/ee/api/graphql/reference/)
* [Everbase docs](https://www.everbase.co/reference/Query)
